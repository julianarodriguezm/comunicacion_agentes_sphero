# Comunicación entre agentes con Spheros
Repositorio para código fuente de programa de comunicación entre agentes con spheros utilizando JavaScript.

## Configuración del entorno
En Sphero Edu, es necesario crear el programa lider y seguidor por separado. Para esto, se pueden crear dos programas diferentes en la aplicación Sphero Edu, uno para el líder y otro para el seguidor. Luego, se pueden copiar y pegar los códigos correspondientes en cada programa., estos se encuentran en los archivos `lider.js` y `seguidor.js` respectivamente.

## Ejecución del programa
Se pueden iniciar ambos programas en la aplicación Sphero Edu, asegurándose de que ambos Spheros estén conectados y listos para comunicarse. El programa líder enviará comandos al programa seguidor, y el seguidor responderá a esos comandos según lo programado. Estos programas están diseñador para que sean utilizados en la aplicación Sphero Edu.

## Creación de programa
Para crear un programa en Sphero Edu, sigue estos pasos:
1. Abre la aplicación Sphero Edu en tu dispositivo.
2. Conecta tu Sphero a la aplicación.
3. Ve a la sección de programación y selecciona "Crear nuevo programa".
4. Elige el tipo de programa y sphero ("JavaScript" y "Sphero Bolt" respectivamente).
5. Copia y pega el código correspondiente al programa líder o seguidor en el editor de código.
6. Guarda el programa y ejecútalo para ver cómo los Spheros se comunican entre sí.
